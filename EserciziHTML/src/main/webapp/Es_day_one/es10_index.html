Esercizi Modulo 2 Web Tier - HTML e CSS - Day 1

Per svolgere gli esercizi, creare nella propria cartella di progetto una sottocartella chiamata "Es_day_one". Per ogni esercizio creare un file html con il nome così formato: es*_index.html, avendo cura di sostituire il carattere * con il numero dell'esercizio svolto all'interno.

  1. Creare una pagina HTML5 vuota e aggiungere un titolo, una paragrafo e un'immagine
  2. Creare una tabella con un header, alcune righe e colonne dati.
  3. Creare un modulo di contatto con alcuni campi di input (nome, email, messaggio). Inserisci anche un pulsante "invia". Al click del pulsante, far apparire un alert con scritto "modulo di contatto inviato".
  4. Creare un elenco di elementi non ordinati e un elenco di elementi ordinati.
  5. Creare un elenco di elementi ordinati, che parte dalla numerazione 17 , all'interno di un elenco di elementi non ordinati.
  6. Crea una pagina con un'immagine a sinistra e un testo a destra
  7. Crea una pagina con un'immagine sotto l'altra
  8. Crea una pagina con un'immagine e del testo su di essa
  9. Creare una pagina con un header, un footer e un contenuto principale (attento a non confondere l'header con l'head della pagina visto a lezione
  10. Crea una pagina html che contiene questa traccia di esercizi. I punti della lista devono essere messi in una lista ordinata. Ogni elemento della lista è un hyperlink composto dal testo della traccia. Cliccando sul link, deve aprirsi nel browser la pagina html con la soluzione.
